# feedcat

Build me `feedcat`, a small command-line tool in **Rust**.

It should fetch web feeds, keep a local cache so re-running is fast, and let me
read recent items from the terminal. Make it something I'd actually reach for day
to day. Get it building and the core covered by tests.

That's the whole brief. Start when ready.
